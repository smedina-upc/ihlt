# Learning PCFGs I

### Requirements

```python3
import nltk
nltk.download('treebank')
from nltk.corpus import treebank
from nltk import PCFG, ViterbiParser
!pip install svgling

```

### DataSet

```python3
productions = []
S = nltk.Nonterminal('S')
for f in treebank.fileids():
    for tree in treebank.parsed_sents(f):
        productions += tree.productions()
```
