# Lesk in NLTK

**Requirements**:

```python3
import nltk
nltk.download('wordnet')
nltk.download('omw-1.4')
```

**Use**:

```python3
context = ['I', 'went', 'to', 'the', 'bank', 'to', 'deposit', 'money', '.']

synset = nltk.wsd.lesk(context, 'bank', 'n')
```

